📚 Aprende Arrays

🗄️ ¿Qué es un arreglo?

Un arreglo (o array) es una estructura de datos que almacena múltiples valores en una sola variable, ordenados en posiciones numeradas.

Piensa en él como un conjunto de lockers numerados: cada locker guarda exactamente un valor y tiene un número único (su índice).

💡 Analogía: los lockers

[0]🍎
[1]42
[2]"hola"
[3]vacío

Cada celda del arreglo es un "locker" que puede contener cualquier valor: número, texto, emoji, etc.

📝 Cómo crear un arreglo

// Arreglo vacío let lockers = []; // Con valores iniciales let frutas = ["manzana", "pera", "uva"]; // También funciona con: let nums = new Array(5); // 5 espacios

🔢 Índices: la dirección de cada locker

Los arreglos en programación empiezan desde 0, no desde 1. El primer elemento siempre está en el índice 0.

[0]🐶
[1]🐱
[2]🐭
[3]🐹
let animales = ["🐶", "🐱", "🐭", "🐹"]; // Leer un elemento: console.log(animales[0]); // "🐶" console.log(animales[2]); // "🐭" // ¡Cuidado! Índice fuera de rango: console.log(animales[5]); // undefined

📏 Longitud del arreglo

La propiedad .length te dice cuántos elementos tiene:

let colores = ["rojo", "verde", "azul"]; console.log(colores.length); // 3 // Último elemento: let ultimo = colores[colores.length - 1]; // "azul"

✏️ Modificar elementos

let lockers = ["A", "B", "C"]; // Guardar un valor en el índice 1: lockers[1] = "Nuevo"; // lockers → ["A", "Nuevo", "C"] // Leer el valor del índice 0: let v = lockers[0]; // "A"

🔄 Recorrer con for

let frutas = ["🍎", "🍊", "🍋"]; // Con índice: for (let i = 0; i < frutas.length; i++) { console.log(i, frutas[i]); } // Con for...of: for (let fruta of frutas) { console.log(fruta); }

🛠️ Métodos esenciales

let nums = [10, 20, 30]; // Agregar al final: nums.push(40); // [10,20,30,40] // Quitar del final: nums.pop(); // [10,20,30] // Agregar al inicio: nums.unshift(5); // [5,10,20,30] // Quitar del inicio: nums.shift(); // [10,20,30] // Buscar un elemento: nums.indexOf(20); // 1 // Verificar si incluye: nums.includes(30); // true

🔀 Métodos avanzados

let vals = [1, 2, 3, 4, 5]; // Filtrar pares: let pares = vals.filter(x => x % 2 === 0); // [2, 4] // Transformar (×2): let dobles = vals.map(x => x * 2); // [2, 4, 6, 8, 10] // Sumar todos: let suma = vals.reduce((acc, x) => acc + x, 0); // 15
